How to use Textmill

Open a document

Drop a PDF anywhere on the start screen, use Open PDF…, or paste a link with Open URL…. URL loading only works when the hosting site permits cross-site access; if a link fails, download the file and drop it in instead.

Build parts

A part is a named selection of pages that becomes its own PDF. Click + Add part, then click pages to toggle them in the active part. Shift-click selects a range. A page can belong to several parts at once, and parts don't need to be contiguous.

Read pages up close

Double-click a page to open the reader. Space toggles the page in the active part, ←/→ turn pages, +/− or ⌘/Ctrl+wheel zoom, Esc closes.

Find text

Type in the search box, then Enter for the next match and Shift+Enter for the previous one. Matching pages are highlighted in the grid. Scanned pages without a text layer won't produce matches.

Export

Choose One PDF per part, a Combined PDF, or both, optionally set a base name, then Build PDF(s). Files go to your Downloads folder unless you pick an output folder. The Files panel gives you From/To folder panes for browsing sources and destinations directly.

Requirements & privacy

Works best in Chrome or Edge — the folder features rely on the File System Access API, which Safari and Firefox don't support (exports fall back to Downloads there). All processing happens in your browser: no document data ever leaves your machine.
